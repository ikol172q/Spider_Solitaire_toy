## 构建前检查清单（一次性安装成功）

在运行 `buildozer android debug` 之前，确认以下事项：

1. **buildozer.spec 关键配置**
   - `android.accept_sdk_license = True` ← 自动接受 SDK license
   - `android.archs = arm64-v8a` ← Mate 40 架构
   - `android.uses_legacy_storage = 1` ← Android 10+ scoped storage 兼容
   - `requirements = python3,kivy==2.3.1` ← 依赖完整（不要加 pillow）
   - `p4a.branch = develop` ← p4a 版本与 Kivy 2.3.1 兼容
   - `android.presplash_color = #1a661a` ← 启动时绿色背景（非白屏）

2. **字体文件**
   - `assets/fonts/chinese.ttf` 存在且 > 1MB
   - `source.include_exts` 包含 `ttf`

3. **存储路径**
   - stats.py 使用 `Context.getExternalFilesDir()` 获取 Android 外部目录
   - 不直接写 `/sdcard/`（Android 10+ scoped storage 限制）
   - main.py 中 `Window.size` 只在桌面平台设置（IS_ANDROID 检测）

4. **依赖环境**
   ```bash
   # 确认 buildozer 和 cython 已安装
   pip install buildozer cython
   # 确认 Java JDK 已安装（buildozer 需要）
   java -version  # 应输出 11 或 17
   ```

5. **清理旧构建（首次或遇问题时）**
   ```bash
   buildozer android clean
   rm -rf .buildozer/
   buildozer android debug
   ```
